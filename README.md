PHP Vagrant Development Box
=======================================

Installation
------------

* Install vagrant (http://www.vagrantup.com/)
* Clone or download this repository
* Run ```sudo vagrant up``


Installed Components
--------------------

* Centos 6.3 x64
* Apache 2.2 (including mod_ssl)
* PHP 5.4
* MySQL 5.5
* phpMyAdmin 4.0
